# Embedded_Project

## 개요

- 리눅스 환경 아래에서 시스템 콜 함수를 통해 산업 재해 상황을 관측할 수 있는 멀티 쓰레드 프로그램을 구현했다.

</br>

- 가속도 계 값을 받아 실시간 그래프를 구현해 사용자가 알기 쉽게 지진계의 상태를 관측할 수 있다.

## 구현 기능

- **가속도계 값을 통한 실시간 그래프 구현**
- 버튼을 통한 메뉴 구현
- 가속도계 센서를 통해 진동 감지 후 위험 상황 표시
- 온도 센서를 통해 온도 감지 후 위험 상황 표시
